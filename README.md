# Company Profit Analysis Dashboard

## Project Overview
This project is an interactive Excel dashboard built to analyze company purchase and profit trends using data visualization and KPI tracking.

## Features
- KPI Cards for quick business insights
- Category-wise purchase analysis
- Seasonal revenue trends
- Gender-based purchase comparison
- Discount impact analysis
- Interactive slicers for filtering
- Pivot Tables & Pivot Charts

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Analysis
- Dashboard Design

## Key Insights
- Clothing category generated highest purchase amount
- Spring/Winter showed strong seasonal revenue
- Purchase trends vary across customer segments
